# 🧩 Problem #2574 – Left and Right Sum Differences

**Difficulty:** Easy   
**Topics:** `Prefix Sum`, `Array`     
**Link:** [Leetcode](https://leetcode.com/problems/left-and-right-sum-differences)

---

## 📖 Problem Summary

For every index `i`:

* `leftSum[i]` = sum of all elements before index `i`
* `rightSum[i]` = sum of all elements after index `i`

Return:

```text
answer[i] = |leftSum[i] - rightSum[i]|
```

---

## 💡 Intuition

For each position, we need information from:

```text
Left Side
Right Side
```

A naive approach would recalculate both sums for every index:

```text
O(n²)
```

Instead, we can precompute:

* Left sums
* Right sums

and answer every index in `O(1)`.

---

## 🎯 Key Observation

For every index:

```text
answer[i]
=
|leftSum[i] - rightSum[i]|
```

If we already know:

```text
leftSum[i]
rightSum[i]
```

then computing the answer becomes trivial.

## ✅ Why This Works

For every index:

```text
leftSum[i]
```

contains the sum of elements before `i`.

and

```text
rightSum[i]
```

contains the sum of elements after `i`.

Thus:

```text
|leftSum[i] - rightSum[i]|
```

matches the problem definition exactly.

---

## 💻 Code

```ts
function leftRightDifference(nums: number[]): number[] {
    const n = nums.length;

    const leftSum = new Array(n);

    let sum = 0;
    for(let i = 0; i < n; i++){
        leftSum[i] = sum;
        sum += nums[i];
    }

    const rightSum = new Array(n);

    sum = 0;
    for(let i = n - 1; i >= 0; i--){
        rightSum[i] = sum;
        sum += nums[i];
    }

    const ans = new Array(n);

    for(let i = 0; i < n; i++){
        ans[i] = Math.abs(leftSum[i] - rightSum[i]);
    }

    return ans;
}
```

## ⚡ Approach 2 — Space Optimized Prefix Sum

### Key Observation

We don't actually need both arrays.

If we know:

```text
totalSum
```

then:

```text
rightSum
=
totalSum - leftSum - nums[i]
```

---

### Example

```text
nums = [10,4,8,3]

totalSum = 25
```

At index `1`:

```text
leftSum = 10

rightSum
=
25 - 10 - 4
=
11
```

So:

```text
answer
=
|10 - 11|
=
1
```

No suffix array required.

## 💻 Code

```ts
function leftRightDifference(nums: number[]): number[] {
    const totalSum = nums.reduce((sum, num) => sum + num, 0);

    let leftSum = 0;

    const ans = new Array(nums.length);

    for(let i = 0; i < nums.length; i++){
        const rightSum = totalSum - leftSum - nums[i];

        ans[i] = Math.abs(leftSum - rightSum);

        leftSum += nums[i];
    }

    return ans;
}
```

## 📝 Key Takeaways

* Prefix sums efficiently provide left-side information.
* Suffix sums efficiently provide right-side information.
* Many prefix/suffix problems can later be optimized using a total sum.
* Always check whether auxiliary arrays are truly required.

---

## ⚠️ Common Mistakes

### Recomputing Left and Right Sums

For every index:

```text
Calculate left side
Calculate right side
```

would lead to:

```text
O(n²)
```

---

### Including Current Element

Remember:

```text
leftSum[i]
```

contains elements strictly before `i`.

and

```text
rightSum[i]
```

contains elements strictly after `i`.

The current element must not be included.

---

### Forgetting Absolute Difference

The answer requires:

```text
|leftSum[i] - rightSum[i]|
```

not:

```text
leftSum[i] - rightSum[i]
```

which can produce negative values.
