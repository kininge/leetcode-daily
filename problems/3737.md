# 🧩 Problem #3737 – Count Subarrays With Majority Element I

**Difficulty:** Medium  
**Topics:** `Array`, `Prefix Sum`, `Math`, `Fenwick Tree`   
**Link:** https://leetcode.com/problems/count-subarrays-with-majority-element-i/   

---

## 💡 Intuition

At first glance, the problem looks like:

```text
For every subarray

↓

Count target frequency

↓

Check Majority
```

This naturally leads to a brute-force solution.

But after solving a few examples, we notice that the target count can be carried forward while extending a subarray.

Even better, the majority condition itself can be transformed into a Prefix Sum problem.

```text

```



```text
O(n log n)
```

## 🧠 Why +1 / -1 Works

Suppose a subarray contains:

```text
Target = 5

Others = 3
```

Majority means:

```text
5 > 3
```

Now convert:

```text
Target → +1

Others → -1
```

Subarray sum becomes:

```text
(+5) + (-3)

=

2
```

Positive.

---

Another example:

```text
Target = 2

Others = 4
```

Sum:

```text
(+2) + (-4)

=

-2
```

Negative.

Therefore:

```text
Subarray Sum > 0

⇔

Target is Majority
```

This observation transforms the problem into a Prefix Sum problem.

## ✅ My Solution

### Approach 1 — Brute Force

```ts
function countMajoritySubarrays(nums: number[], target: number): number {
    const n = nums.length;

    function isMajorityElement(startIndex: number, endIndex: number): boolean {
        const _n = endIndex-startIndex+1;
        const half = _n/2;
        let count = 0;
        for(let i=startIndex; i<=endIndex; i++){
            if(nums[i] === target) count++;
            if(count > half) break;
        }

        // console.log(startIndex, "-", endIndex, '-->', count > half);
        return (count > half);
    }

    let subArrayCount: number = 0;
    for(let i=0; i<n; i++){
        for(let j=i; j<n; j++){
            if(isMajorityElement(i, j)) subArrayCount++;
        }
    }
    
    return subArrayCount;
};
```

Time:

```text
O(n³)
```

### Approach 2 — Carry Running Count

```ts
function countMajoritySubarrays(nums: number[], target: number): number {
    const n = nums.length;

    let subArrayCount: number = 0;

    for(let i=0; i<n; i++){
        let targetCount = 0;

        for(let j=i; j<n; j++){
            if(nums[j] === target) targetCount++;

            if(targetCount * 2 > (j-i+1))
                subArrayCount++;
        }
    }

    return subArrayCount;
}
```

### Approach 3 — Prefix Sum Transformation

```ts
function countMajoritySubarrays(nums: number[], target: number): number {
    const n = nums.length;

    let carry = 0;

    for(let i=0; i<n; i++){
        if(nums[i] === target)
            carry++;
        else
            carry--;

        nums[i] = carry;
    }

    let subArrayCount = 0;

    for(let i=0; i<n; i++){
        for(let j=i; j<n; j++){
            const previousPrefix =
                (i===0)?0:nums[i-1];

            const subArraySum =
                nums[j]-previousPrefix;

            if(subArraySum > 0)
                subArrayCount++;
        }
    }

    return subArrayCount;
}
```
