# 🧩 Problem #2812 – Find the Safest Path in a Grid

**Difficulty:** Medium    
**Topics:** `Graph`, `Matrix`, `Breadth First Search`, `Dijkstra`, `Heap`    
**Link:** https://leetcode.com/problems/find-the-safest-path-in-a-grid/    

---

## 💡 Intuition

This problem actually consists of **two completely different graph problems**.

### Step 1

For every cell, calculate:

```text
Distance to the nearest thief
```

This creates a new grid:

```text
Grid
        ↓
Safeness Grid
```

Example:

```text
4 3 2
5 4 1
6 5 0
```

Now every cell has a **safeness value**.

---

### Step 2

Now ignore the thieves.

The problem becomes:

```text
Start
      ↓
Reach End
      ↓
Choose a path whose
minimum safeness
is as large as possible.
```

This is **not** a shortest path problem.

It is a:

```text
Maximum Bottleneck Path
```

problem.

## 🚀 Evolution of Thought Process

### ❌ First Attempt

```text
Create Safeness Grid
        ↓
Greedy DFS
        ↓
Always choose safer neighbour
```

Wrong.

Local optimum does not guarantee a globally safest path.

---

### ❌ Second Attempt

```text
Create Safeness Grid
        ↓
DFS
        ↓
Explore paths
```

This results in exponential exploration and TLE.

---

### ✅ Final Observation

The grid has now become a graph.

```text
Grid
      ↓
Weighted Graph
```

Each cell has a weight:

```text
Safeness Value
```

Goal:

```text
Maximize

minimum weight

along the path
```

This is exactly a Dijkstra-style problem using a **Max Heap**.


```text

```

Continue expanding the safest frontier first.

Eventually the minimum safeness encountered on the chosen path becomes the answer.

## ✅ My Solution

```ts
function maximumSafenessFactor(grid: number[][]): number {
    const n = grid.length;
    // edge case
    if(grid[0][0] === 1 || grid[n-1][n-1]) return 0;

    // now we need to create safeness matrix
    const safenessGrid = [];
    for(let i=0; i<n; i++) safenessGrid.push(new Array(n).fill(500));

    // Multi-Source BFS
    const queue = new MyQueue();
    const visisted = new Set();

    for(let i=0; i<n; i++){
        for(let j=0; j<n; j++){
            if(grid[i][j] === 1){
                visisted.add(`${i}-${j}`);
                queue.append({x:i, y:j, safe:0});
            }
        }
    }

    const row = [-1,0,1,0];
    const col = [0,1,0,-1];

    while(queue.peak()){
        const node = queue.pop();
        const {x,y,safe} = node as {x:number,y:number,safe:number};

        safenessGrid[x][y] = safe;

        for(let i=0;i<4;i++){
            const _x=x+row[i];
            const _y=y+col[i];
            const key=`${_x}-${_y}`;

            if(_x>=0&&_x<n&&_y>=0&&_y<n&&!visisted.has(key)){
                visisted.add(key);
                queue.append({
                    x:_x,
                    y:_y,
                    safe:safe+1
                });
            }
        }
    }

    // Max Heap (Dijkstra Variant)
    const comparator=(a,b)=>a.safe>b.safe;
    const maxHeap=new MyHeap(comparator);

    visisted.clear();

    visisted.add("0-0");

    maxHeap.append({
        x:0,
        y:0,
        safe:safenessGrid[0][0]
    });

    let minSafeCell=500;

    while(maxHeap.peak()){

        const node=maxHeap.pop();

        const {x,y,safe}=node as {
            x:number,
            y:number,
            safe:number
        };

        minSafeCell=Math.min(minSafeCell,safe);

        if(x===n-1&&y===n-1) break;

        for(let i=0;i<4;i++){

            const _x=x+row[i];
            const _y=y+col[i];

            const key=`${_x}-${_y}`;

            if(_x>=0&&_x<n&&_y>=0&&_y<n&&!visisted.has(key)){

                visisted.add(key);

                maxHeap.append({
                    x:_x,
                    y:_y,
                    safe:safenessGrid[_x][_y]
                });

            }
        }
    }

    return minSafeCell;
}
```

## ⚖️ Approach Comparison

| Approach                    | Time        | Space | Notes               |
| --------------------------- | ----------- | ----- | ------------------- |
| Greedy DFS                  | ❌           | O(n²) | Local optimum fails |
| DFS Search                  | Exponential | O(n²) | TLE                 |
| Multi-Source BFS + Max Heap | O(n² log n) | O(n²) | Optimal             |

## ⏱️ Complexity Analysis

Let

```text
n = grid.length
```

### Multi-Source BFS

Every cell is visited once.

```text
O(n²)
```

---

### Max Heap Traversal

Each cell is inserted once.

Heap operations:

```text
O(log n²)

=

O(log n)
```

Overall:

```text
O(n² log n)
```

---

### Space Complexity

```text
Safeness Grid

+

Visited

+

Queue

+

Heap
```

Total:

```text
O(n²)
```

