# 🧩 Problem #2095 – Delete the Middle Node of a Linked List

**Difficulty:** Medium  
**Topics:** `Linked List`, `Two Pointers`   
**Link:** [Delete the Middle Node of a Linked List](https://leetcode.com/problems/delete-the-middle-node-of-a-linked-list/)

---

## 💡 Intuition

To delete the middle node, we first need to find it.

A brute-force approach would be:

```text
Count Length
    ↓
Find Middle Index
    ↓
Traverse Again
    ↓
Delete Middle
```

But we can do better.

Using Fast & Slow pointers:

```text
slow → moves 1 step

fast → moves 2 steps
```

When `fast` reaches the end:

```text
slow
```

will be standing on the middle node.

The only extra thing we need is:

```text
prev
```

which tracks the node before the middle so we can remove it.

```text

```

```text
4 → 1
```

```text

```

## ✅ My Solution

```ts
function deleteMiddle(head: ListNode | null): ListNode | null {
    // edge case 
    if(head.next === null) return null;

    let prev: ListNode | null = null;
    let slow: ListNode | null = head;
    let fast: ListNode | null = head;

    while(fast && fast.next){
        prev = slow;
        slow = slow.next;
        fast = fast.next.next;
    }

    // remove middle node
    prev.next = slow.next;

    return head;
};
```
