# 🧩 Problem #1189 – Maximum Number of Balloons

**Difficulty:** Easy   
**Topics:** `Hash Map`, `String`   
**Link:** [Maximum Number of Balloons](https://leetcode.com/problems/maximum-number-of-balloons/)  

---

## 💡 Intuition

To build the word:

```text
balloon
```

we need:

```text
b → 1
a → 1
l → 2
o → 2
n → 1
```

The question becomes:

```text
How many complete "balloon" words
can be formed from available characters?
```

The answer is determined by the character that runs out first.

```text
Available Characters
        ↓
Required Characters
        ↓
Find Limiting Character
```

### Final Observation

```text
balloon

b → 1
a → 1
l → 2
o → 2
n → 1
```

Since:

```text
l
o
```

appear twice, we must divide their frequencies by:

```text
2
```

before computing the answer.

---

## 🧠 Dry Run

### Example

```text
text = "loonbalxballpoon"
```

Frequency:

```text
b → 2
a → 2
l → 4
o → 4
n → 2
```

Required:

```text
b → 2/1 = 2
a → 2/1 = 2
l → 4/2 = 2
o → 4/2 = 2
n → 2/1 = 2
```

Answer:

```text
min(2,2,2,2,2)
=
2
```

## ✅ My Solution

```ts
function maxNumberOfBalloons(text: string): number {
    const store = new Map();

    // O(n)
    for(const char of text){
        if(store.has(char)){
            store.set(char, store.get(char)+1);
        } else {
            store.set(char, 1);
        }
    }

    let count = 0;

    if(store.has("b") && store.has("a") && store.has("l") && store.has("o") && store.has("n")){
        count = store.get("b");
        count = Math.min(count, store.get("a"));
        count = Math.min(count, Math.floor(store.get("l")/2));
        count = Math.min(count, Math.floor(store.get("o")/2));
        count = Math.min(count, store.get("n"));
    }

    return count;
};
```

## 🔄 Alternative Approach

### My Approach (Hash Map)

| Metric | Value |
| ------ | ----- |
| Time   | O(n)  |
| Space  | O(1)  |

Uses a frequency map to count characters.

---

### Alternative (Frequency Array)

Since:

```text
Only lowercase English letters
```

we can use:

```ts
const freq = new Array(26).fill(0);
```

instead of a map.

| Metric | Value |
| ------ | ----- |
| Time   | O(n)  |
| Space  | O(1)  |

Slightly faster due to direct indexing.

---

## ⚖️ Approach Comparison

| Approach        | Time | Space | Notes                        |
| --------------- | ---- | ----- | ---------------------------- |
| Hash Map        | O(n) | O(1)  | Flexible                     |
| Frequency Array | O(n) | O(1)  | Preferred for fixed alphabet |
