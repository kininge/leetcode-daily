# 🧩 Problem #1732 – Find the Highest Altitude

**Difficulty:** Easy    
**Topics:** `Array`, `Prefix Sum`   
**Link:** [Find the Highest Altitude](https://leetcode.com/problems/find-the-highest-altitude/)   

---

## 💡 Intuition

The biker starts at:

```text
Altitude = 0
```

Each value in `gain` tells us how much altitude changes between two points.

```text
Current Altitude
        +
Gain/Loss
        ↓
Next Altitude
```

We don't actually need to store all altitudes.

Instead:

```text
Keep Current Altitude
        +
Track Maximum Altitude Seen So Far
```

That's enough to solve the problem in one pass.

## ✅ My Solution

```ts
function largestAltitude(gain: number[]): number {
    let highestAltitude = 0;
    let altitude = 0;

    for(const height of gain){
        altitude += height;
        highestAltitude = Math.max(highestAltitude, altitude);
    }

    return highestAltitude;
};
```

### Alternative

Build the complete altitude array first.

```text

↓
-5
↓
-4
↓
```

Then find the maximum.

| Metric | Value |
| ------ | ----- |
