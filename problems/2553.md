# 🧩 Problem #2553 – Separate the Digits in an Array

**Difficulty:** Easy  
**Topics:** `Array`, `Math`, `Simulation`, `String`  
**Link:** https://leetcode.com/problems/separate-the-digits-in-an-array/

---

## 📝 Problem Summary

You are given an array of positive integers `nums`.

For every integer in the array, separate all of its digits while preserving their original order.

Return a new array containing all separated digits in the same sequence they appear.

---

### Example

Input:

```txt
nums = [13,25,83,77]
``` 

### Digit separation:

```txt
13 → [1,3]
25 → [2,5]
83 → [8,3]
77 → [7,7]
```

### Final result:

```txt
[1,3,2,5,8,3,7,7]
```

### Key Observation

Each number can be processed independently.

There are two common ways to separate digits:

1. Convert the number to a string and iterate through characters
2. Use mathematical digit extraction (`% 10` and `/ 10`)

Both approaches work efficiently for the given constraints.

### Constraints

- `1 <= nums.length <= 1000`
- `1 <= nums[i] <= 10^5`

---

## 💡 Intuition

The problem is straightforward:
for every number in the array, extract its digits and append them to the final answer while maintaining their original order.

---

### Approach 1 — String Conversion

A simple way is to:
1. Convert the number into a string
2. Iterate through each character
3. Convert the character back into a number
4. Push it into the answer array

Example:

```txt
83
→ "83"
→ ['8', '3']
→ [8, 3]
```
This approach is short, clean, and easy to understand.

---

### Approach 2 — Mathematical Digit Extraction

We can also extract digits mathematically using: `num % 10`

to get the last digit and: `Math.floor(num / 10)`

to remove the last digit.

However, digits are extracted in reverse order:
```txt
83
→ 3
→ 8
```

So we temporarily store extracted digits and later insert them back in reverse order to preserve the original sequence.

---

### Why Both Work

Each number is processed independently, and every digit is added exactly once to the final array.

Since the maximum number size is small (`<= 10^5`), both approaches are efficient enough.

---

## 🛠 Approach 1 — String Conversion

### Idea

Convert each number into a string and iterate through its characters.

Each character represents a digit, so we convert it back into a number and append it to the answer array.

---

### Step 1 — Traverse the Array

Process every number independently:

```ts
for (const num of nums)
```

### Step 2 — Convert Number to String

Convert the number into a string: `num.toString()`
Example: `83 → "83"`

### Step 3 — Extract Digits

Iterate through every character: `'8', '3'`

Convert each character back into a number and push it into the answer array.

---

### Why It Works

String conversion preserves the original digit order automatically.

Since each digit is processed exactly once, the solution remains efficient and easy to implement.

---

### Code (TypeScript)

```ts
function separateDigits(nums: number[]): number[] {
    const answer: number[] = [];

    for (const num of nums) {
        for (const digit of num.toString()) {
            answer.push(Number(digit));
        }
    }

    return answer;
}
```

---

## 🛠 Approach 2 — Mathematical Digit Extraction

### Idea

Instead of converting numbers into strings, we can extract digits mathematically.

For a number: `num`
- `num % 10` gives the last digit
- `Math.floor(num / 10)` removes the last digit

---

### Problem With Direct Extraction

Digits are extracted in reverse order.

Example:
```txt
83
→ 3
→ 8
```

But required order is: `[8, 3]`

So we:

1. Store extracted digits temporarily
2. Traverse them in reverse order
3. Append them into the final answer array

---

### Step 1 — Extract Digits

Repeatedly extract the last digit: `digits.push(num % 10)`
and remove it: `num = Math.floor(num / 10)`

### Step 2 — Restore Original Order

Since digits were collected in reverse order, iterate backward through the temporary array and append digits into the answer array.

### Why It Works

Every digit is extracted exactly once.

Reversing the temporary digit sequence restores the original order of digits in the number.

This approach avoids string conversion and uses only mathematical operations.

---

### Code (TypeScript)

```ts
function separateDigits(nums: number[]): number[] {
    const ans: number[] = [];

    for (const num of nums) {

        let current = num;

        const digits: number[] = [];

        // extract digits in reverse order
        while (current > 0) {
            digits.push(current % 10);
            current = Math.floor(current / 10);
        }

        // restore original order
        for (let i = digits.length - 1; i >= 0; i--) {
            ans.push(digits[i]);
        }
    }

    return ans;
}
```

## ⏱ Complexity Analysis

Let:

```txt
d = total number of digits across all numbers
```

| Approach | Time Complexity | Auxiliary Space | Notes |
|----------|----------------|-----------------|-------|
| String Conversion | `O(d)` | `O(d)` | Uses temporary string representations |
| Mathematical Digit Extraction | `O(d)` | `O(1)` | Uses arithmetic operations and temporary digit storage |
