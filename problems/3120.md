# 🧩 Problem #3120 – Count the Number of Special Characters I

**Difficulty:** Easy    
**Topics:** `String`, `Hash Map`    
**Link:** https://leetcode.com/problems/count-the-number-of-special-characters-i/

---

## 📝 Problem Summary

A letter is called:
```txt
special
```

if it appears in:
* lowercase
* uppercase

inside the given string.

Return:
```txt
count of special characters
```

### Example

Input:
```txt
"aaAbcBC"
```

Characters:
```txt
a + A
b + B
c + C
```
Answer:
```txt
3
```

### 💡 Key Observation

Each English letter has:
```txt
2 versions
```

Example:
```txt
a ↔ A
b ↔ B
```

A character becomes special when:
```txt
both cases exist
``` 

---

## 🛠 Approach 1 — Single Pass HashSet

### 🔥 Core Idea

While traversing the string:

store already-seen characters inside:
```txt
Set
```

For current character:

* compute opposite case
* check whether opposite case already exists

If yes:

* current character becomes special

### ⚠️ Important Trick

This condition:

```ts
!store.has(letter)
```

prevents:
```txt
duplicate counting
```

### Example

Input:
```txt
"aAaAaA"
```

Without duplicate prevention:
```txt
A would be counted multiple times
```

Your condition guarantees:
```txt
each letter counted only once
```

### Step-by-Step Example

Input:
```txt
"aaAbcBC"
``` 

Process `a`
```txt
store = {a}
count = 0
```

Process second `a`

Opposite:
```txt
A
```

not found.

No count.

Process `A`

Opposite:
```txt
a
```

exists.

AND:
```txt
A not already processed
```

So:
```txt
count++
```

Now:
```txt
count = 1
```

### Continue

Same happens for:
```txt
b/B
c/C
```

Final:
```txt
count = 3
```
---

## 💻 Code (TypeScript)
```ts
function numberOfSpecialChars(
    word: string
): number {

    const store = new Set();

    let count = 0;

    for (const letter of word) {

        const isSmallCase =
            letter === letter.toLowerCase();

        const oppositeCase =
            isSmallCase
                ? letter.toUpperCase()
                : letter.toLowerCase();

        // special character found
        // avoid duplicate counting
        if (
            store.has(oppositeCase) &&
            !store.has(letter)
        ) {
            count++;
        }

        store.add(letter);
    }

    return count;
}
``` 
---

### ✅ Why This Works

Suppose current letter is:
```txt
A
```

If:
```txt
a already exists
```

then:
```txt
a/A pair confirmed
```

We count it once and store current character.

Future occurrences won't increase count again.

## 🛠 Approach 2 — Two HashSets

### 💡 Idea

Maintain:

* one set for lowercase letters
* one set for uppercase letters

Then count common letters.

### Step-by-Step

Input:
```txt
"aaAbcBC"
```

### Lowercase Set
```txt
{a,b,c}
```

### Uppercase Set

Convert uppercase letters to lowercase:
```txt
{a,b,c}
```

### Common Letters
```txt
a,b,c
```

Count:
```txt
3
```

---

## 💻 Code (TypeScript)

```ts
function numberOfSpecialChars(
    word: string
): number {

    const lower = new Set();
    const upper = new Set();

    for (const ch of word) {

        if (ch === ch.toLowerCase()) {
            lower.add(ch);
        }
        else {
            upper.add(ch.toLowerCase());
        }
    }

    let count = 0;

    for (const ch of lower) {

        if (upper.has(ch)) {
            count++;
        }
    }

    return count;
}
```

## 🔥 Which Approach Is Better?

First approach is actually more interesting because:

* single pass
* no second traversal
* avoids maintaining two structures

The two-set solution is slightly easier for beginners to reason about.

| Operation |	Complexity |
|---|---|

Why Space Is `O(1)`?

Because maximum possible stored characters:
```txt
26 lowercase + 26 uppercase
```

Only:
```txt
52 characters
```

constant size.
