### Key observation

For any integer `x`:

- `x % 3 == 0` â†’ already good â†’ cost `0`
- `x % 3 == 1` â†’ nearest multiple of 3 is `x - 1` â†’ cost `1`
- `x % 3 == 2` â†’ nearest multiple of 3 is `x + 1` â†’ cost `1`

    0 if num % 3 == 0
    1 otherwise

**Algorithm**

  1. Initialise steps = 0.
  2. For each num in nums:
    - If num % 3 != 0, increment steps by 1.
  3. Return steps.

    let steps = 0;

    }
