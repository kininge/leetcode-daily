### Minute Hand

A full circle is:

```text
360°
```

```text
60 minutes
```

```text
360 / 60 = 6°
```

per minute.

---

### Hour Hand

A full circle is:

```text
360°
```

There are:

```text
12 hours
```

So:

```text
360 / 12 = 30°
```

per hour.

---

### Important Observation

At:

```text
3:30
```

the hour hand is NOT:

```text
90°
```

It has already moved halfway toward 4.

So we must account for the minute contribution.

```text
Hour Hand
    ↓
Base Hour Angle
    +
Minute Movement
```

## ✅ My Solution

```ts
function angleClock(hour: number, minutes: number): number {
    // consider 12 as 0 degree
    // angle between each hour is 30 degree
    // represent hour and minute hand in degree start from 12 or 0

    // base degree
    let hourDegree: number = (hour%12)*30;
    // due to minute hand some movement in hour hand as well
    hourDegree += (minutes/60)*30;
    const minuteDegree = (minutes*6)%360

    const largeDegree = Math.max(hourDegree, minuteDegree);
    const smallDegree = Math.min(hourDegree, minuteDegree);

    const gap1 = largeDegree-smallDegree;
    const gap2 = 360-largeDegree+smallDegree;
    return Math.min(gap1, gap2);
};
```

## 🔄 Alternative Approach

### My Approach

| Metric | Value |
| ------ | ----- |
| Time   | O(1)  |
| Space  | O(1)  |

Uses explicit angle calculations and computes both possible circular distances.

---

### Simplified Version

Instead of:

```ts
const largeDegree = Math.max(hourDegree, minuteDegree);
const smallDegree = Math.min(hourDegree, minuteDegree);

const gap1 = largeDegree-smallDegree;
const gap2 = 360-largeDegree+smallDegree;

return Math.min(gap1, gap2);
```

We can directly write:

```ts
const diff = Math.abs(hourDegree - minuteDegree);
return Math.min(diff, 360 - diff);
```

Same logic.

Cleaner code.
