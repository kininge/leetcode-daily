## ğŸ›  Approach  

We use **Dynamic Programming (DP)**:  

1. If `matrix[i][j] == 0` â†’ no square ends here.  
2. Otherwise:  
   $$dp[i][j] = \min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1$$
   (with boundaries treated as `0`).  

This formula works because:  
- A new square can only form if top, left, and top-left cells also had squares.  
- Taking the `min` ensures we only extend as far as the smallest neighbor allows.  

ğŸ‘‰ Each cellâ€™s `dp[i][j]` tells us: *â€œHow many different square sizes end at this cellâ€*.  
So we just sum them all up.

```javascript []
var countSquares = function(matrix) {
    let total = 0;

    for (let i = 0; i < matrix.length; i++) {
        for (let j = 0; j < matrix[0].length; j++) {
            if (matrix[i][j] === 1 && i > 0 && j > 0) {
                matrix[i][j] = Math.min(
                    matrix[i - 1][j], 
                    matrix[i][j - 1], 
                    matrix[i - 1][j - 1]
                ) + 1;
            }
            total += matrix[i][j];
        }
    }
    return total;
};
```
