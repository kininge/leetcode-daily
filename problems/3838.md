# 🧩 Problem #3838 – Weighted Word Mapping

**Difficulty:** Easy   
**Topics:** `String`, `Simulation`, `Array`   
**Link:** [Weighted Word Mapping](https://leetcode.com/problems/weighted-word-mapping/)   

---

## 💡 Intuition

Each character contributes a predefined weight.

For every word:

```text
Word
    ↓
Sum character weights
    ↓
Take modulo 26
    ↓
Map using reverse alphabet
```

The only interesting observation is the reverse mapping:

```text
0  → z
1  → y
2  → x
...
25 → a
```

Instead of creating a lookup table, we can directly calculate the character using ASCII values.

## ✅ My Solution

```ts
function mapWordWeights(words: string[], weights: number[]): string {
    let ans = '';
    const A = 'a'.charCodeAt(0);
    const Z = 'z'.charCodeAt(0);

    for(const word of words){
        let num = 0;

        for(const char of word){
            const index = char.charCodeAt(0) - A;
            num += weights[index];
        }

        num %= 26;
        ans += String.fromCharCode(Z - num);
    }

    return ans;
};
```
